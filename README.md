Durante a atividade, criei um algoritmo no Flowgorithm utilizando símbolos como início, entrada de dados, processamento, decisão e saída de dados. Esses símbolos ajudam a organizar a lógica do programa e facilitam o entendimento de como as informações são processadas.

Exercício 1

Crie um fluxograma que:

Receba o nome de uma pessoa digitado pelo usuário

Mostre na tela a mensagem: “Olá, [nome da pessoa]!”

Exercício 2

Crie um fluxograma que:

Leia dois números informados pelo usuário

Faça o cálculo da soma desses números

Mostre na tela o resultado da soma

Exercício 3

Crie um fluxograma que:

Leia a idade de uma pessoa

Caso a idade seja maior ou igual a 18, mostrar a mensagem “Maior de idade”

Caso contrário, mostrar “Menor de idade”

Exercício 4

Crie um fluxograma que:

Leia um número digitado pelo usuário

Verifique e informe se o número é positivo, negativo ou igual a zero

Exercício 5

Crie um fluxograma que:

Receba duas notas

Faça o cálculo da média entre elas

Mostre na tela o valor da média final
